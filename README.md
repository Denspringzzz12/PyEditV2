# PyEdit IDE - Kivy版本

一个真正跨平台的Python IDE，支持Android、iOS、macOS、Windows和Linux。

## 版本对比

### V1 (Flet版本) vs V2 (Kivy版本)

| 特性 | V1 (Flet) | V2 (Kivy) |
|------|-----------|-----------|
| **GUI框架** | Flet (基于Flutter) | Kivy (原生Python) |
| **架构** | 混合渲染 | 原生OpenGL渲染 |
| **部署方式** | 需要运行时 | 完全原生应用 |
| **跨平台** | ✅ | ✅ |
| **性能** | 中等 | 优秀 |
| **移动端支持** | 良好 | 优秀 |
| **触摸优化** | 良好 | 优秀 |
| **离线运行** | ✅ | ✅ |
| **包大小** | 较大 | 较小 |
| **系统集成** | 有限 | 良好 |

## V2 新特性

1. **真正的原生体验**
   - 每个平台都是原生应用
   - 更好的性能表现
   - 完整的系统集成

2. **优化的移动端体验**
   - 专为触摸屏设计
   - 响应式布局
   - 移动端手势支持

3. **更好的性能**
   - OpenGL硬件加速
   - 更少的内存占用
   - 更快的启动速度

4. **部署优势**
   - Android: 直接生成APK
   - iOS: 生成iOS应用
   - 桌面: 可执行文件

# PyEdit IDE

一个基于Python的轻量级集成开发环境，支持代码编辑、运行和终端操作。

## 版本对比

### V1 (Flet版本) vs V2 (NiceGUI版本)

| 特性 | V1 (Flet) | V2 (NiceGUI) |
|------|-----------|--------------|
| **GUI框架** | Flet | NiceGUI |
| **架构** | 客户端-服务器 | Web应用 |
| **部署方式** | 桌面/移动应用 | Web服务 |
| **跨平台** | ✅ | ✅ |
| **实时更新** | ❌ | ✅ |
| **代码热重载** | ❌ | ✅ |
| **访问方式** | 本地应用 | 浏览器访问 |
| **移动端支持** | ✅ | ✅ (响应式) |
| **依赖管理** | 较重 | 较轻 |
| **开发体验** | 良好 | 优秀 |

## V2 新特性

1. **Web原生架构**
   - 通过浏览器访问
   - 更好的跨平台兼容性
   - 无需安装客户端

2. **改进的开发体验**
   - 代码热重载
   - 实时UI更新
   - 更好的调试支持

3. **现代化UI**
   - 响应式设计
   - 更好的移动端支持
   - 现代化的组件库

4. **部署灵活性**
   - 支持本地部署
   - 支持服务器部署
   - 支持Docker容器化

## 功能特性

- ✅ 用户认证系统（注册/登录）
- ✅ Python代码编辑器
- ✅ 代码执行和输出显示
- ✅ 集成终端
- ✅ 代码补全支持
- ✅ 多平台支持
- ✅ 响应式UI设计
